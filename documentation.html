Résumé

Ce projet consiste à développer une application de gestion de bibliothèque intégrant un assistant conversationnel. L'application permet de gérer un catalogue de livres (CRUD complet) et d'interagir avec un chatbot capable de comprendre des requêtes en langage naturel. L'objectif est de moderniser l'accès à l'information documentaire tout en offrant une expérience utilisateur fluide et intuitive.

Mots-clés : Bibliothèque, Python, SQLite, Chatbot, Intelligence artificielle, Gestion de données.

Base de données

Table principale : livres

ChampTypeDescription
id_livreINTEGERIdentifiant unique (auto-incrémenté)
titreTEXTTitre du livre
auteurTEXTNom de l'auteur
categorieTEXTGenre littéraire
annee_publicationINTEGERAnnée de parution
quantite_disponibleINTEGERExemplaires disponibles
statutTEXTdisponible / emprunté / réservé
date_empruntTEXTDate d'emprunt
date_retour_prevueTEXTDate de retour estimée
emprunteurTEXTNom de l'emprunteur

Exemple de requête

-- Récupérer tous les livres disponibles
SELECT * FROM livres WHERE statut = 'disponible';

-- Emprunter un livre
UPDATE livres 
SET quantite_disponible = quantite_disponible - 1,
    statut = 'emprunté',
    date_emprunt = '04/06/2026',
    date_retour_prevue = '18/06/2026',
    emprunteur = 'Jean Dupont'
WHERE id_livre = 1 AND quantite_disponible > 0;

Fonctionnement du chatbot

Le chatbot utilise l'API Google Gemini pour interpréter les questions en langage naturel. Il est capable de :

  • 🔍 Rechercher un livre par ID, titre ou auteur
  • 📖 Vérifier la disponibilité d'un ouvrage
  • 💡 Faire des recommandations personnalisées
  • 💬 Converser naturellement avec l'utilisateur

Conclusion

Ce projet a permis de développer une application complète de gestion de bibliothèque, alliant interface graphique moderne, base de données relationnelle et intelligence artificielle.

L'intégration du chatbot Gemini offre une expérience utilisateur naturelle et fluide, tandis que le tableau de bord permet une visualisation claire des données. L'application répond à l'ensemble des objectifs fixés et constitue une base solide pour d'éventuelles évolutions.
